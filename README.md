# Cell-Phone-Dataset-Insights
Cell Phone Dataset  Insights
The goal of this stage is to:

Perform Exploratory Data Analysis (EDA) using Tableau.
Build two interactive Tableau dashboards to analyze the data and support user-driven exploration. 
Ensure the dashboards effectively present key insights, patterns, and trends identified during EDA. 
Clearly explain the business case and analytical outcomes through the data story. 
Prepare a comprehensive business report.
